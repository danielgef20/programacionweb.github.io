Ejercicio 10.7

Indique si cada uno de los siguientes enunciados es verdadero o falso. Si es falso, explique por qué.

  1. Para hacer referencia a una ubicación o elemento específico en un arreglo, especificamos el nombre del arreglo y el valor del elemento.

    2. Falso, se debe indicar el nombre del arreglo y el índice del elemento.

  2. La declaración de una variable reserva espacio para un arreglo.

    3. Verdadero.

  3. Para indicar que deben reservarse 100 ubicaciones para el arreglo entero p, el programador debe escribir la siguiente declaración: p[ 100 ];

    4. Falso, se debe escribir: var p = new Array(100);

  4. Un programa de JavaScript que inicializa los elementos de un arreglo de 15 elementos en cero debe contener cuando menos una instrucción for.

    5. Falso, para poder asignar los valores no es necesario un ciclo for, pero se utiliza cuando el arreglo es grande.

  5. Un programa de JavaScript que calcula el total de los elementos de un arreglo bidimensional debe contener instrucciones for anidadas.

    6. Verdadero, si se refiere al total de los valores si puede ser necesario, si se refiere al total de elementos que tiene el arreglo no es necesario.
